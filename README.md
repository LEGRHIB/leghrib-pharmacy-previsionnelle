# Leghrib Pharmacy Previsionnelle

Gestion previsionnelle des stocks pour officine, basee sur l'import Excel (nomenclature, ventes mensuelles, rotation, base DCI) avec tableaux de bord, alertes, achats, fournisseurs et peremption.

## Quick start

1. Open pharmplanV3.html in a browser.
2. Import your XLSX files (nomenclature, monthly, rotation, national DCI).
3. Run the forecast.

## Project structure

- pharmplanV3.html: main HTML entry
- styles.css: extracted styles
- app.js: extracted application logic

## Notes

- The app runs fully client-side.
- Data stays in the browser (localStorage for settings).
